## Overview
This repository contains the complete submission for the AI Ethics assignment on *Designing Responsible and Fair AI Systems*. It demonstrates theoretical understanding, case study analysis, practical fairness auditing using the COMPAS dataset, and ethical reflections. The project emphasizes responsible AI practices, bias detection, and mitigation strategies.

---

---

## Contents

### Part 1: Theoretical Understanding
- Covers definitions and explanations of:
  - Algorithmic Bias
  - Transparency vs Explainability
  - GDPR Impact on AI
  - Ethical Principles: Justice, Non-Maleficence, Autonomy, Sustainability

### Part 2: Case Study Analysis
- Biased Hiring Tool (Amazon AI)
  - Source of bias, mitigation strategies, and fairness metrics
- Facial Recognition in Policing
  - Ethical risks and responsible deployment policies

### Part 3: Practical Audit
- Python/Jupyter Notebook using:
  - AI Fairness 360
  - Pandas
  - Matplotlib & Seaborn
  - Scikit-learn
- Conducts bias evaluation on the COMPAS dataset
- Computes and visualizes:
  - Disparate Impact Ratio
  - False Positive Rate Difference
  - Statistical Parity Difference
- Includes bias mitigation and post-mitigation evaluation

### Part 4: Ethical Reflection
- Personal or future project reflection
- Ensures adherence to ethical AI principles
- Covers fairness, transparency, privacy, and accountability measures

## How to Run

1. Clone the repository:

git clone <repository_url>

2. Navigate to the project folder:

cd ai-ethics-assignment


3. Install dependencies:

pip install -r requirements.txt


4. Open the Jupyter Notebook:

jupyter notebook COMPASS.ipynb


5. Run cells sequentially to generate metrics and visualizations.
